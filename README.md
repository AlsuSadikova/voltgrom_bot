Телеграм бот предназначен для для автоматизации информирования клиентов по часто задаваемым вопросам путём общения с пользователями в мессенджере.
  - Предоставляет общую информацию о фирме, услугах и гарантиях. 
  - Показывает ассортимент товаров
  - Даёт возможность клиентам предварительно выбрать стабилизатор, благодаря функции подсчёта мощности как по приборам, так и по силе тока(реализовано с использованием API стороннего сайта).
  - Собирает и отправляет информацию администраторам, если клиент хочет оставить заявку на обратную связь.
